Public Function:download(url, outputPath)
Impact: Tải file âm thanh từ URL và lưu dưới dạng .mp3 tại server.
Input: url: string, outputPath: string
Output: Promise<void>
Micro-Logic:
  • Sử dụng promisify(exec) để chạy lệnh shell không đồng bộ.
  • Thực thi lệnh: yt-dlp -x --audio-format mp3 -o "path" "url"
  • -x: Extract audio only.
  • --audio-format mp3: Ép định dạng đầu ra.
* Throw InternalServerErrorException nếu yt-dlp trả về mã lỗi hoặc không tìm thấy URL.

Public Function:cleanup(filePath)
Impact: Xóa file vật lý đã tải về để dọn dẹp dung lượng đĩa.
Input: filePath: string
Output: Promise<void>
Micro-Logic:
  • Kiểm tra sự tồn tại của file bằng fs.existsSync.
  • Xóa file đồng bộ bằng fs.unlinkSync.
  • Catch lỗi nội bộ và ghi log, không ném lỗi ra ngoài để tránh làm gãy luồng chính.
