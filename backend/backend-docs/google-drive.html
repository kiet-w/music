Module: google-drive (Micro-Technical Detail)

TRÁCH NHIỆM CHÍNH

Tích hợp API Google Drive để liệt kê và tải file âm thanh (MP3, WAV, FLAC...) từ Drive cá nhân và Shared Drives của người dùng thông qua Google OAuth Token.

DEPENDENCIES
  • googleapis (v3)
  • StorageService (Internal)
  • SongRepository (Internal)

Controller Handler

listFiles

Lấy danh sách các file audio từ Drive của người dùng. Bao gồm try/catch để trả về chi tiết lỗi cho frontend.

IN / OUT

IN: Query: token (string) | OUT: Array<DriveFile> | ErrorObject

graph TD
Start[GET /files] --> CallSvc[Service.listFiles]
CallSvc --> Success{Success?}
Success -- Yes --> Res[Return Files List]
Success -- No --> Catch[Catch Error & Log]
Catch --> ResErr[Return Detailed Error Object]
Controller Handler

importFile

Quy trình tải file từ Drive, xử lý tên file, upload lên Storage và lưu vào DB.

IN / OUT

IN: ImportDto | OUT: Song (Database Record)

Lấy Metadata từ Drive để biết tên gốc và mimeType.
Tải file content (Buffer).
Sanitize Name: Bỏ dấu tiếng Việt, thay ký tự đặc biệt bằng _.
Upload lên Supabase Storage qua StorageService.
Tạo record Song trong DB (giữ nguyên tên gốc làm Title).
graph TD
Start[POST /import] --> Meta[Get Metadata]
Meta --> Down[Download Buffer]
Down --> Sanitize[Sanitize Filename]
Sanitize --> Upload[StorageService.upload]
Upload --> DB[SongRepository.create]
DB --> End[Return Song]
Public Service

listFiles

Liệt kê file audio, hỗ trợ Shared Drives và xử lý Shortcut.

MICRO-LOGIC
Dùng supportsAllDrives: true để truy cập Shared Drives.
Lọc file theo mimeType (audio/*) hoặc extension (.mp3, .wav...).
Map các Shortcut về targetId và đánh dấu isShortcut: true.
graph TD
Init[OAuth2.setCredentials] --> API[Drive.files.list]
API --> Filter[Filter Audio & Shortcuts]
Filter --> Map[Resolve Shortcut IDs]
Map --> End[Return Resolved Files]
Public Service

getFileMetadata

Lấy thông tin cơ bản (id, name, mimeType, size) của một fileId cụ thể.

graph LR
In[fileId] --> OAuth[Auth Client]
OAuth --> API[Drive.files.get]
API --> Out[Return Metadata Object]
Public Service

downloadFile

Tải nội dung file nhị phân từ Google Drive.

MICRO-LOGIC
Sử dụng alt: 'media' để lấy content.
Thiết lập responseType: 'arraybuffer' để nhận binary data.
graph TD
In[fileId] --> Auth[Auth Client]
Auth --> API[Drive.files.get alt:media]
API --> Buf[Buffer.from ArrayBuffer]
Buf --> End[Return Buffer]
