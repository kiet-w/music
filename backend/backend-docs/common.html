HttpExceptionFilter
src/common/filters/http-exception.filter.ts

Bắt và format lại các lỗi `HttpException` thành cấu trúc JSON chuẩn.

graph LR
E[Exception] --> F[Filter Catch]
F --> G[Extract Status]
F --> H[Extract Message]
G --> I[Format Response]
H --> I
I --> J[Response.json]
Micro-Logic
  • Sử dụng `@Catch(HttpException)` để đăng ký filter.
  • Trích xuất `status` từ `exception.getStatus()`.
  • Phân tích `exception.getResponse()` để lấy thông báo lỗi chi tiết nhất.
  • Trả về JSON bao gồm: `statusCode`, `message`, `timestamp`, `path`.
