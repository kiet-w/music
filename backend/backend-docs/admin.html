Trách nhiệm

Cung cấp các API quản trị cấp cao để thao tác trực tiếp với cơ sở dữ liệu bài hát và hệ thống lưu trữ Cloud (Supabase/S3). Không chứa logic nghiệp vụ phức tạp mà đóng vai trò là "orchestrator".

async deleteTrack(@Param('id') id: string) {
  return this.songRepository.delete({
    where: { id },
  });
}
cleanupStorage(@Body() body: CleanupStorageBody)
POST
INPUT (IN)
{
  bucketName: string,
  path: string
}
OUTPUT (OUT)
{
  message: "Storage cleanup initiated",
  file: string
}
Impact: Kích hoạt tiến trình dọn dẹp file không mong muốn trên Cloud Storage. Đảm bảo tính nhất quán giữa Storage và Database. 
graph TD
A[Request Body] --> B[AdminController.cleanupStorage]
B --> C{StorageCleanupService.cleanupFile}
C --> D[Storage Provider API]
D --> E[Cloud Storage File Deleted]
B --> F[Return Success Message]
MICRO-LOGIC 
@HttpCode(HttpStatus.OK)
async cleanupStorage(@Body() body: { bucketName: string; path: string }) {
  await this.storageCleanupService.cleanupFile(body.bucketName, body.path);
  return { message: 'Storage cleanup initiated', file: body.path };
}
